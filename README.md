# 🎵 Yoow - Site Institucional de Artista

Projeto desenvolvido para um artista, com objetivo de centralizar sua presença digital e facilitar o acesso ao seu conteúdo.

## 🚀 Funcionalidades

- Página institucional responsiva
- Integração com Spotify
- Links para redes sociais
- Layout moderno com foco em experiência do usuário

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 📱 Responsividade

O site foi desenvolvido para funcionar em dispositivos desktop e mobile.

## 🎯 Objetivo do projeto

Praticar desenvolvimento front-end e integração com plataformas externas, simulando um cenário real de cliente.
